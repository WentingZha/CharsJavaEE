What is Servlet?

>

A Java applet running on the website

Send requests and receive responses

  1. The servlets implement the Servlet Class
  2. The servlets extend from GenericServlet or HttpServlet
  3. The servlets implement JspPage and HttpJspPage

Servlet is a single instance

init(): Only run on the the first request is received

request: doGet()/doPost() -> destroy()

How to write servlet。Extend javax.servlet.http.HttpServlet

Overwrite doGet() and doPost()
